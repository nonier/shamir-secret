
Функционал работает через REST запросы на локальный сервер
Приложение запускается на 80 порту

Поддерживаемые методы:

POST: /secret
Создает и разбивает секрет на части
В параметрах передается JSON в виде:

{
    "secret": 12345,        //значение секрета
    "partCount": 5,         //количество частей
    "p": 4                  //поле, в котором будут создаваться ключи
}

GET: /secret
Пытается восстановить секрет из входных частей
В параметрах передается JSON в виде:
{
[
    {
    "point": 12,            //значение x
    "value": 2345,          //значение y
    "p": 4                  //поле, в котором создался ключ
    },
    {
    "point": 33,
    "value": 4212,
    "p": 4
    }
]
}
